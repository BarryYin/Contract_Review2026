# AI智能合同合规审查工具黑客松：合同测试包（主办方模拟版）

本测试包为模拟样本，仅供参赛团队进行文档解析、OCR、条款结构化、双语一致性分析、规则引擎和风险评分演示使用，不构成真实法律文件。

## 文件清单

1. `01_中文采购框架协议_含表格附件盖章扫描件.docx`
   - 约20页，含基础信息表、价格表、技术附件、交付排程、模拟签署页扫描件、模拟盖章图片。
   - 预设风险：付款周期90日、违约金5%/日、单方变更与解除、保密期限1年、知识产权归属过宽、管辖地偏向甲方。

2. `02_中英双语IT服务合同_双栏对照.docx`
   - 约15页，采用中英双栏对照表格排版。
   - 预设风险：付款周期75日、合理时间等模糊表达、客户单方变更、责任上限不对等、无理由终止、语言效力条款中英文冲突。

3. `03_标准保密协议NDA_英文原版.docx`
   - 约5页，英文原版NDA。
   - 预设风险：保密义务存续期18个月，低于2年；返还销毁期限为reasonable time，存在模糊表达。

4. `04_含手写补充条款的劳动合同_混合OCR.docx`
   - 约8页，正文为可解析文本，末页含模拟手写补充条款图片。
   - 预设风险：岗位/地点可单方调整、奖金调整口径宽泛、竞业限制补偿另行约定、手写补充含高额违约金与过宽竞业限制。

## 建议评测字段

- 合同类型、合同编号、签约方、期限、金额、付款周期、争议解决、保密期限、知识产权归属、终止条款。
- 日期、金额、地名、公司名称、联系人、身份证号、邮箱/电话等实体。
- 双语合同需输出中英文条款对照，并识别第13条语言效力冲突。
- OCR场景需识别采购合同签署页扫描件与劳动合同手写补充条款。
